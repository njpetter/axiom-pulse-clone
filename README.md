# Axiom Pulse Clone


**Tech:** Next.js 14 (App Router) + TypeScript (strict) + Tailwind CSS + Redux Toolkit + React Query + Radix UI


This project is a pixel-faithful token discovery table inspired by Axiom Trade Pulse. It includes real-time mocked price updates, popovers, tooltips, modals, skeletons, visual-regression tests and CI.
